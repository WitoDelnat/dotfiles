# dotfiles

Configuration of tools and dotfiles.

## Usage

**Brewfile**

```
brew bundle install
```

**Npmfile**

```
< Npmfile.txt xargs npm install -g
```

**K9s**

```
cp -r k9s ~/.k9s
```

**Vim**

```
cp vim/.vimrc ~/.vimrc
```

**Zsh**

```
cp zsh/.zshrc ~/.zshrc
```

**iTerm2**

```
cp iterm2 ~/.iterm2
```

Afterwards configure _Preferences > General > Preferences tab > Load preferences from a customfolder_ with _~/.iterm2/_.
